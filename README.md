# 3D Viewer

A browser-based GLB/GLTF 3D model viewer built with Three.js.

## Features

- Drag-and-drop or file picker to load `.glb` models
- Orbit controls for rotating, panning, and zooming
- Dark minimal UI
- No build step required — single `index.html` file

## Usage

Open `index.html` in a modern browser, then drag a `.glb` file onto the viewer or click to browse.

## Tech

- [Three.js](https://threejs.org/) for 3D rendering
- Vanilla HTML/CSS/JS — no framework dependencies
